Python Final Projects

A collection of beginner-friendly Python GUI applications built with Tkinter. These projects were developed as final practical projects to demonstrate core Python programming concepts, GUI development, API integration, file handling, and problem-solving skills.


Projects Included

1. Quiz / Exam System

An interactive quiz application that allows users to answer multiple-choice questions and receive a final score at the end of the quiz.

Features
	•	Welcome screen
	•	Student name input
	•	Multiple-choice questions
	•	Real-time score calculation
	•	Final result summary
	•	Restart quiz functionality
	•	Save scores to file
	•	GUI built with Tkinter

Concepts Used
	•	Functions
	•	Lists and dictionaries
	•	Conditional statements
	•	Loops
	•	File handling
	•	Tkinter GUI programming

2. Weather Application

A weather application that fetches and displays real-time weather data using the OpenWeather API￼.

Features
	•	Search weather by city name
	•	Display temperature
	•	Display humidity
	•	Display weather condition
	•	API integration using Requests
	•	Error handling with try/except
	•	Press Enter to fetch weather
	•	GUI built with Tkinter

Concepts Used
	•	API integration
	•	JSON data handling
	•	Exception handling
	•	HTTP requests
	•	Tkinter GUI programming


Technologies Used
	•	Python
	•	Tkinter
	•	Requests
	•	OpenWeather API￼

Move into the project folder:
cd python_final_project

Install independencies:
pip install requests

Running the Applications:
Quiz Application:

python main.py

Weather Application:
cd weather_app
python main.py

API Setup for Weather App
	1.	Create an account at OpenWeather￼
	2.	Generate your API key
	3.	Add your API key inside config.py
